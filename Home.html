Comandos Flex

Lista con los comandos de Flex CSS y una breve explicacion:

  • display: flex;
    : Esta propiedad se aplica al contenedor principal para activar el modelo de caja flexible (Flexbox) en sus elementos hijos directos. Esto permite que los elementos hijos se coloquen y se ajusten dinámicamente dentro del contenedor de acuerdo con las reglas de Flexbox. 
  • flex-direction: row | row-reverse | column | column-reverse;
    : Esta propiedad establece la dirección principal en la que los elementos hijos se colocan dentro del contenedor flexible. Por ejemplo, row los coloca en una fila de izquierda a derecha, column los coloca en una columna de arriba hacia abajo, y las variantes reverse invierten la dirección. 
  • justify-content: flex-start | flex-end | center | space-between | space-around | space-evenly;
    : Esta propiedad controla la alineación de los elementos a lo largo del eje principal del contenedor flexible. Permite distribuir el espacio sobrante entre los elementos de diferentes maneras, como centrarlos, alinearlos al inicio o al final, o distribuir el espacio uniformemente entre ellos. 
  • align-items: flex-start | flex-end | center | baseline | stretch;
    c: Esta propiedad controla la alineación de los elementos a lo largo del eje transversal del contenedor flexible. Se utiliza para alinear los elementos verticalmente dentro del contenedor. Por ejemplo, flex-start alinea los elementos en la parte superior, center los centra verticalmente, stretch los estira para que ocupen todo el espacio vertical, etc. 
  •  flex-wrap: nowrap | wrap | wrap-reverse;
    : Esta propiedad controla si los elementos hijos del contenedor flexible deben envolverse en múltiples líneas o no cuando el espacio disponible no es suficiente para colocar todos los elementos en una sola fila o columna. 
  • align-content: flex-start | flex-end | center | space-between | space-around | stretch;: Esta propiedad
                se utiliza para controlar cómo se distribuye el espacio adicional a lo largo del eje transversal cuando
                hay múltiples líneas de elementos dentro del contenedor flexible. Funciona de manera similar a
                justify-content, pero en lugar de aplicarse a elementos individuales, se aplica al espacio entre las
                líneas de elementos.

Comandos Grid

comandos de Grid CSS y una breve explicacion:

  • display: grid;
    : Esta propiedad se aplica al contenedor principal para activar el modelo de diseño de cuadrícula en sus elementos hijos directos. 
  • grid-template-columns y grid-template-rows
    : Estas propiedades se utilizan para definir el tamaño y el número de columnas y filas en la cuadrícula respectivamente. Puedes especificar tamaños fijos, tamaños fraccionarios (fr), tamaños automáticos (auto), etc. 
  • grid-gap o gap
    : Estas propiedades se utilizan para especificar el espacio entre las filas y columnas de la cuadrícula. grid-gap define espacios entre filas y columnas por separado, mientras que gap es una abreviatura que establece ambos al mismo valor. 
  • grid-template-areas
    : Esta propiedad permite definir áreas en la cuadrícula y asignar nombres a esas áreas. Luego, puedes colocar elementos hijos dentro de estas áreas utilizando la propiedad grid-area en los elementos hijos. 
  • justify-items y align-items
    : Estas propiedades se utilizan para alinear los elementos hijos dentro de las celdas de la cuadrícula a lo largo del eje principal (justify-items) y el eje secundario (align-items) respectivamente. 
  • justify-content y align-content
    : Estas propiedades se utilizan para alinear y distribuir el contenido de la cuadrícula a lo largo del eje principal (justify-content) y el eje secundario (align-content) respectivamente./li> 
  • grid-auto-columns y grid-auto-rows
    : Estas propiedades se utilizan para especificar el tamaño de las columnas y filas automáticas, que se aplican a las celdas que no están explícitamente definidas en la cuadrícula. 
  • grid-auto-flow
    : Esta propiedad especifica cómo se deben colocar los elementos que no tienen una posición explícita definida dentro de la cuadrícula. Puedes establecerlo en row, column o dense.
